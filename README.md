# Black Spotify

Application developed during the year 2019. Shows music albums from [Spotify](https://www.spotify.com).

## What the project has

- Hosting on **Netlify**
- Spotify auth via **PKCE** (no backend required)
- Sentry for error tracking
- Redux
- Webpack
- Progressive Web App
- React
- [Spotify Web API](https://developer.spotify.com/documentation/web-api/)
- Cypress for E2E tests
- Cucumber (BDD) for Cypress
- Mocha and Chai for unit tests

## Getting started

See [DEVELOPMENT.md](./DEVELOPMENT.md) for full setup instructions.

## How to contribute

Fork the repository, create a branch from `master`, make your changes, and open a pull request back to `master`.

### Where help is needed

- Redux sometimes returns the last request instead of the current one
- Responsiveness improvements
- Animations
- Bug report button
